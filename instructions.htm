This page is meant to be an instruction manual for using my moving planner.
Come with me on this great adventure!


First: This web application is not meant to be an end all in the research you should take when planning a move as your circumstances will differ from mine and anyone elses.

Planner: This is a page with a list that you will create the many different tasks that you need to undertake in order for your move to go smoothly!
Repairs: This page is meant to outline the kinds of repairs and cleaning that one should do when moving out of an apartment.
Utilities: This page show you a short list of utilities that you may have to disconnect, however this is not all encompassing as some apartments have more or less utilities than others. Make sure you check with your landlord if there is anything else you need to cancel or disconnect.
Supplies: This is a page with a list of companies that will have all of the supplies you need when moving.
